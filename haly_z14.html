Sposoby definiowania kolorów

Sposób Przykład Opis
Nazwa koloru color: red; Kolor zapisany słownie
HEX color: #FF0000; Kod szesnastkowy koloru
RGB color: rgb(255,0,0); Mieszanie kolorów czerwony, zielony, niebieski
RGBA color: rgba(255,0,0,0.5); RGB + przezroczystość
HSL color: hsl(0,100%,50%); Barwa, nasycenie, jasność
HSLA color: hsla(0,100%,50%,0.5); HSL + przezroczystość
